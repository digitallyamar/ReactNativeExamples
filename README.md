# Working Offline

## 3 ways to store data offline
1. AsyncStorage - 6 MB limit, Key-Value pairs
2. SecureStore - Provided by Expo, 12 MB limit, Key-Value pairs and encrypted so can store sensitive info here.
3. SQLite - Query tables
